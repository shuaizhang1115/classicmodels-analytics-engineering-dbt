# GitHub upload steps

```bash
cd classicmodels_analytics_showcase
git init
git add .
git commit -m "Add dbt analytics engineering showcase project"
git branch -M main
git remote add origin https://github.com/<your-username>/classicmodels-analytics-engineering.git
git push -u origin main
```

```text
classicmodels-analytics-engineering-dbt
```

dbt analytics engineering project that transforms raw order-management data into a tested star schema and BI-ready OBT on PostgreSQL.
